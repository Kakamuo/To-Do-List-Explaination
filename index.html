JavaScript Task List Code Explained (Eng + UrduLish)

Code Explanation (Code ki Wazahat)

1. Document Ready Event (Document ka load hona)

English: This code waits for the document to fully load before executing the JavaScript. The `DOMContentLoaded` event ensures that all HTML elements are ready to be used.

UrduLish: Ye code is baat ko ensure karta hai ke script tabhi chale jab tak document puri tarah load nahi ho jata. `DOMContentLoaded` event is baat ka confirmation deta hai ke sare HTML elements ready hain use hone ke liye.

Code:

document.addEventListener("DOMContentLoaded", () => { ... });

2. Selecting Elements (Elements ko select karna)

English: We are selecting the elements by their IDs, which allows us to interact with them later. These elements are the input field, the button to add tasks, and the list where tasks will appear.

UrduLish: Hum elements ko unke IDs ke zariye select kar rahe hain taake baad mein hum unse interact kar sakein. Yeh elements hain task input field, task add karne wala button, aur task list jahan tasks dikhayi denge.

Code:

const taskInput = document.getElementById("task-input");
const taskAddBtn = document.getElementById("task-add");
const taskList = document.getElementById("task-list");

3. Adding Task Event (Task add karne ka event)

English: We add a click event listener to the "Add Task" button. This makes sure that whenever the user clicks on it, the `addTask` function is triggered.

UrduLish: Hum "Add Task" button par click ka event listener add karte hain. Is se yeh ensure hota hai ke jab user button par click kare, to `addTask` function run ho jaye.

Code:

taskAddBtn.addEventListener("click", addTask);

4. Add Task Function (Task add karne ka function)

English: The `addTask` function is responsible for checking if the input field is empty. If it’s not, it creates a new task and appends it to the task list.

UrduLish: `addTask` function ka kaam yeh hai ke yeh check kare agar input field khali nahi hai. Agar kuch likha hai, to yeh ek nayi task create karke task list mein add kar deta hai.

Code:

function addTask() {
  const taskText = taskInput.value.trim();

  if (taskText !== "") {
    const taskItem = document.createElement("li");
    taskItem.textContent = taskText;

    taskList.appendChild(taskItem);
    taskInput.value = "";
  } else {
    alert("You must write something");
  }
}

5. Mark Task as Completed (Task ko complete mark karna)

English: When a task item is clicked, the class "completed" is toggled. This can change its style (like a strikethrough) to show it's done.

Urdu-ish: Jab koi task item click hota hai, to us par "completed" class toggle ho jati hai. Is se task ka style change hota hai, jaise ke us par line lag jana, jo yeh dikhata hai ke task complete ho gaya hai.

Code:

taskItem.addEventListener("click", () => {
  taskItem.classList.toggle("completed");
});

6. Deleting Task on Right Click (Right click se task delete karna)

English: When a task is right-clicked, it will be removed from the list. We use `contextmenu` event and call the `remove()` method to delete the task.

Urdu-ish: Jab kisi task par right-click kiya jata hai, to woh list se remove ho jata hai. Hum `contextmenu` event use karte hain aur `remove()` method call karke task ko delete kar dete hain.

Code:

taskItem.addEventListener("contextmenu", (e) => {
  e.preventDefault();
  taskItem.remove();
});

Complete JavaScript Code Example (JavaScript Code ki Puri Misal)

document.addEventListener("DOMContentLoaded", () => {
  const taskInput = document.getElementById("task-input");
  const taskAddBtn = document.getElementById("task-add");
  const taskList = document.getElementById("task-list");

  taskAddBtn.addEventListener("click", addTask);

  function addTask() {
    const taskText = taskInput.value.trim();

    if (taskText !== "") {
      const taskItem = document.createElement("li");
      taskItem.textContent = taskText;

      taskItem.addEventListener("click", () => {
        taskItem.classList.toggle("completed");
      });

      taskItem.addEventListener("contextmenu", (e) => {
        e.preventDefault();
        taskItem.remove();
      });

      taskList.appendChild(taskItem);
      taskInput.value = "";
    } else {
      alert("You must write something");
    }
  }
});
